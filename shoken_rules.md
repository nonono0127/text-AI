# 通知表 所見 文体・構成ルール

Excelファイル（コピー5-1所見.xls）の96件を分析して抽出したルールです。

---

## 1. 基本仕様

| 項目 | 内容 |
|------|------|
| 文字数 | 133〜206字、**目安は160〜175字** |
| 文数 | 3〜6文（最多は**4文**：57件/96件） |
| 書き出し | 全角スペースなし（96件すべて） |
| 文末の基本 | **「〜ことができました。」が圧倒的多数**（全文末の約65%） |
| 否定的表現 | 一切使用しない（すべて肯定的） |
| 生徒名 | 基本的に使用しない |

---

## 2. 全体構造（2ブロック構成）

```
【前半ブロック：1〜3文】生活面・係活動・委員会・行事・日常の様子
         ↓ 切り替えフレーズ
【後半ブロック：1〜3文】学習面・教科での具体的な活動・成長
```

前半と後半はほぼ必ず分かれており、学習面の話題は後半にまとめる。

---

## 3. 前半ブロック（生活面）

### 3-1. 書き出しパターン（頻出順）

| パターン | 例文 |
|---------|------|
| `前期〇〇として、〜` | 前期体育係として、整列の号令や準備運動を大きな声で行うことができました。 |
| `後期〇〇として、〜` | 後期学級代表として、整列の号令をかけることができました。 |
| `〇〇係として〜` | 給食係として毎日忘れることなく配膳準備や給食当番の手伝いができました。 |
| `〇〇委員として〜` | 図書委員になり、バーコードを読み取る仕事をしっかり行うことができました。 |
| `学級代表として〜` | 学級代表として集会に、遅れないようにみんなを並ばせることができました。 |
| 活動名から直接 | 林間学習のスタンツでは班の中の誰よりも大きな声を出し、会場を盛り上げました。 |
| 日常の様子から | 毎日の掃除を一生懸命行うことができました。 |

### 3-2. 前半でよく登場する活動

**係活動**（最多：49件）
- 給食係、体育係、連絡係、配達係、窓・電気係、整頓係、宿題チェック係、お手伝い係 等

**委員会活動**（10件）
- 図書委員、スポーツ委員、広報委員、生き物委員会、環境委員 等

**その他の役割**
- 学級代表（10件）、児童会役員（6件）、通学班班長

**行事・特別活動**
- 林間学習（2件）、どんぐり発表会、競書会、運動会、6年生を送る会

**日常生活**
- 掃除（11件）、休み時間の過ごし方、友達との関わり

### 3-3. 前半の典型文型

```
〔役割/活動〕として、〔具体的な行動〕ことができました。
〔追加エピソード〕〔称賛フレーズ〕。
```

例：
> 前期は体育係として大きな声を出して、整列、体操、ランニングまでをスムーズに行いました。後期は授業記録係になり、授業後に自分からタブレットで黒板の写真を撮り授業の記録を残してくれました。与えられた仕事をきちんと取り組むことができ感心しました。

## 8. 学年別の表記ルール

### 低学年（1〜2年生が中心）
- **全文ひらがな表記**（漢字を一切使わない）
- 読点「、」は通常通り使用

例：
> こくごのどりょくしょうテストにむけて、どりょくしょうプリントをくりかえしれんしゅうしました。ごうかくてんめざしてがんばるすがたがりっぱでした。

### 中学年以上（3年生〜）
- 通常の漢字仮名交じり文

## 9. 具体性に関するルール

- **活動は「何をどのようにした」まで具体的に記述する**
  - NG: 「給食係の仕事をがんばりました。」
  - OK: 「給食係として毎日忘れることなく配膳準備や給食当番の手伝いができました。他の給食係の子と役割分担をし、効率よく給食準備をする姿からは成長を感じました。」

- **学習面は単元名・実験名・作品名などを明記する**
  - NG: 「理科の実験をがんばりました。」
  - OK: 「理科「もののとけ方」では、食塩とミョウバンの溶ける量を増やす実験を班の子と協力して行いました。」

- **称賛フレーズだけでなく、必ず具体的エピソードをセットで入れる**

---

## 10. 禁止事項

- 否定的・注意的な表現（「〜できていません」「〜が苦手です」等）
- 「頑張ってほしい」「改善が必要」等の課題指摘
- 生徒の欠点や問題行動への言及
- 全員に使い回せる抽象的な定型文のみの使用
